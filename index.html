Smart Door Lock System

Project Overview

The Smart Door Lock System is an IoT-based security solution developed using Arduino Uno and RFID technology. The system verifies authorized users using RFID cards and unlocks the door automatically using a servo motor.

Components Used

  • Arduino UNO
  • RFID RC522 Module
  • Ultrasonic Sensor
  • LCD1602 Display with I2C Module
  • Servo Motor
  • Green LED
  • Red LED
  • Buzzer
  • Breadboard and Connecting Wires

Working Principle

When a person comes within the detection range of the ultrasonic sensor, the LCD display shows a message asking the user to verify their identity. The user places an RFID card near the RFID reader.

If the RFID card matches the authorized ID stored in the system, the servo motor rotates and unlocks the door. The green LED turns ON and the buzzer gives a short beep indicating successful authentication.

If an unauthorized card is detected, the red LED blinks and the buzzer sounds an alert indicating access denied.
